UW Co-op Stats V3

This site shows co-op data of a random sample of University of Waterloo students. The data was taken from LinkedIn, and only includes students in the realm of software with 2 or more experiences. Stay tuned for improvements.

Inspired by the old UW co-op stats V2 found here, we decided to create an updated and sortable version.

We hope this site can help you choose your next internship.
